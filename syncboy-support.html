Phosphor Emulator - Support

Getting Started

Phosphor Emulator emulates Game Boy, Game Boy Color, and Game Boy Advance. To play, you need to provide your own ROM files. Import ROMs by tapping the + button in the ROM library or by opening a ROM file from the Files app.

Supported File Types

  • .gb, Game Boy
  • .gbc, Game Boy Color
  • .gba, Game Boy Advance
  • .sgb, Super Game Boy
  • .agb, Game Boy Advance (alternate extension)

Frequently Asked Questions

How do I save my game?
Phosphor Emulator auto-saves when you exit a game so you can resume where you left off. You also get 10 manual save state slots accessible from the in-game menu. In-game saves (battery saves) work automatically for games that support them.

How does Link Cable work?
Open the in-game menu and tap "Link Cable." Both devices must be on the same local network. One device advertises and the other browses for peers. Once connected, the emulated serial link cable handles trading and multiplayer just like real hardware.

How do I play on Apple Watch?
Open the Phosphor Emulator Watch app and select a game. Your iPhone handles the emulation and streams frames to the Watch in real time. The Watch sends back your D-pad and button inputs. Both devices must be paired and in range.

What is AI Play?
AI Play lets a neural agent learn to play games on its own. Toggle it from the in-game menu. The AI learns by watching your inputs and progressively improving. You can take over at any time by touching the controls.

How does the Game Boy Camera work?
When you load a Pocket Camera ROM, Phosphor Emulator uses your device's front camera as the sensor. The camera feed is converted to the Game Boy Camera's 128x112 pixel 4-shade format in real time. Grant camera permission when prompted.

How do I save Game Boy Printer output?
When a game prints to the virtual Game Boy Printer, tap the printer icon in the in-game menu to view printed images. Long-press any image and tap "Save to Photos" to save it to your photo library.

The game runs too fast or too slow.
Check the speed multiplier in the in-game menu. Set it to 1x for normal speed. Higher values fast-forward the emulation.
