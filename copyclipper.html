A clipboard built for working with AI.

Grab code from anywhere, bundle it with context, and paste a clean prompt into Claude, ChatGPT, or anything that takes text.

CopyClipper sits in your menu bar and turns the messy reality of multi-file context into a clean, structured paste. Pick the files, the functions, the diffs that matter, and copy a single, well-formed block ready to share with any AI assistant.

What it does

From scattered files to a single, perfect prompt.

📂

Multi-file pick list

Browse your project and check off the files you want bundled into the next paste.

🧩

Smart formatting

Output gets fenced code blocks, file headers, and language hints, exactly what assistants want to see.

🔍

Just the diff

Paste only what changed instead of the whole file. Useful for review and PR-style prompts.

📋

Templates

Save common prompt shapes, "Refactor this", "Find the bug", "Write tests for", and apply them in one click.

⚙️

Menu bar speed

Lives in the menu bar. Quick keystroke, picker opens, paste is ready before you finish typing the prompt.

🔒

Stays on your Mac

No cloud, no sync, no telemetry. Files are read locally and assembled on your machine.
